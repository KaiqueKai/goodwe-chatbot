# Modelo de Testes — GoodWe ChargeOps Assistant

| Pergunta | Resposta Esperada |
|---|---|
| Quantos veículos podem carregar ao mesmo tempo? | O chatbot deve explicar que a quantidade depende da potência disponível no condomínio e do sistema de balanceamento de carga. |
| Meu carregamento foi interrompido. O que aconteceu? | O chatbot deve informar possíveis causas como pico energético, balanceamento de potência ou falha no carregador. |
| Como funciona o faturamento individual? | O chatbot deve explicar que o consumo pode ser monitorado individualmente para cobrança separada dos moradores. |
| Existe horário recomendado para carregar o veículo? | O chatbot deve orientar sobre horários de menor demanda energética para otimizar o consumo. |
| Posso reservar um carregador compartilhado? | O chatbot deve explicar como funciona o uso compartilhado e disponibilidade dos eletropostos. |
